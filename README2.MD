# Unsupervised Learning: Clustering Techniques

## Project Overview
This project demonstrates three popular clustering algorithms using the Mall Customer Segmentation dataset.

## Algorithms Used
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN

## Dataset
Mall Customer Segmentation Dataset

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- scipy

## Project Structure
README.md
requirements.txt
notebooks/clustering_analysis.ipynb
src/preprocessing.py
src/clustering_models.py
docs/comparison_report.md

## Results
The clustering algorithms were compared using:
- Silhouette Score
- Davies-Bouldin Index

K-Means produced clear customer groups. Hierarchical Clustering showed similar grouping with a dendrogram. DBSCAN detected clusters and identified noise points.
